![image](https://github.com/user-attachments/assets/793afad3-f5fb-470a-8212-b93563b46115)# BlindMap - 盲人导航地图应用

一个基于 React + TypeScript 的盲人导航地图应用，提供语音输入和实时摄像头导航功能。

## 功能特性

- 📍 实时地图显示
- 🎤 语音输入目的地
- 📹 摄像头实时导航
- 🗺️ 基于 OpenStreetMap 的地图服务

## 技术栈

- React 19
- TypeScript
- Vite
- Leaflet
- Tailwind CSS

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
